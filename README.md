# Processamento de Imagens 

Projeto abrangente de processamento digital de imagens utilizando OpenCV e Pillow. Implementa técnicas avançadas como detecção de bordas, binarização, normalização e manipulação de histogramas.

## Sobre o Projeto

Este código demonstra um pipeline completo de processamento de imagens focado em impressões digitais, aplicando múltiplas técnicas para análise e melhoramento de imagens digitais.

##  Funcionalidades

### 1. **Carregamento e Conversão**
   - Download de imagens da web
   - Conversão entre formatos (Pillow ↔ OpenCV)
   - Transformação para tons de cinza

### 2. **Detecção de Bordas**
   - Filtro Gaussiano para suavização
   - Algoritmo Canny para detecção de bordas
   - Visualização comparativa

### 3. **Binarização**
   - Método automático de Otsu
   - Conversão para imagem preto e branco
   - Limiarização adaptativa

### 4. **Manipulação de Histograma**
   - Normalização de intensidades
   - Equalização para melhoria de contraste
   - Análise comparativa de distribuições

### 5. **Visualização Profissional**
   - Comparação lado a lado
   - Gráficos de histogramas
   - Interface clara e informativa
